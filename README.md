# IfeelgameFramework

这是一个简单的游戏基础框架，旨在缩短项目的初始阶段（框架搭建阶段）周期，快速进入游戏内容的开发阶段，使我们可以专注于游戏逻辑的开发。

**目前的演示代码都放在Demo.cs脚本里，star数超过100的话我会考虑出个文档**
